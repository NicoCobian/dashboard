Events7 Dashboard

Tecnologies: Reactjs, Nestjs, Typescript, Axios, sqlite.


How to run the App;

- run first the front end (events7-frontend) open a terminal over the folder and type : npm start (the front will start in the localhost3000)

- then run the backend (backsql) open a terminal over the folder and type : npm start (the backend will start in the localhost3001)
